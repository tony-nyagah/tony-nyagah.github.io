# Complexity is the enemy

I sit at my desk looking at a class that implements a function that fetches data from a table in a database which in turn triggers updates to another table in the same database if certain conditions are met.  
I am perplexed, angry and frustrated. How did I get here? What is going on? Should I quit?  
This has been my frame of mind since the start of 2024. I was left as the sole developer on a Python/JavaScript codebase and I am slowly being driven into insanity.
